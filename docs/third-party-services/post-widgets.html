AddToAny

{% tabs AddToAny %}

Edit {% label primary@NexT config file %} to enable AddToAny Share:

# AddToAny Share. See: https://www.addtoany.com
addtoany:
  enable: true
  buttons:
    - facebook
    - twitter

After AddToAny enabled, you can add more buttons.

# AddToAny Share. See: https://www.addtoany.com
addtoany:
  enable: true
  buttons:
    - facebook
    - twitter
    - telegram
    - wechat
    - qzone
